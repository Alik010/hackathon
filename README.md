## 
Модель: RT-DETR

### Порядок запуска демо


1. Создание и активация окружения
    ```bash
    python -m venv venv
    . venv/bin/activate 
   или
    . venv/Scripts/activate 
    ```

2. Установка библиотек
   ```
    pip install -r requirements.txt
   ```
   
3. Запуск демо
   ```
   streamlit run demo/app.py
   ```
